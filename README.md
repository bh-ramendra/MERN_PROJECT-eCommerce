# MERN_PROJECT-eCommerce
This is MERN Based eCommerce Application,Contain Admin Pannel So That Admin Can Create Category,Update Category,Check All Categoies, Create Product, Update Product,check all products. after that User can Add Products In Cart. this application with stripe payment integration so that user can pay online and get deliver at his own place.
